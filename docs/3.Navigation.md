# Navigation

You can inject your routes from your own modules into the admin navigation as shown below


    'navigation' => array(
        'admin' => array(
            'mynavigation' => array(
                'label' => 'My Module',
                'route' => 'zfcadmin/myroute',
            ),
        ),
    ),
